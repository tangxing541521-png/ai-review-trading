- 生成趋势核心池。
- 建立模拟买卖、持仓、收益统计和日报。

- 收紧趋势核心池过滤条件。
- 修复模拟账户资产计算。
- 新增交易明细日志。
- 新增策略统计文件。

## Phase 3：收益跟踪

- 增加资金曲线文件。

- 新增历史回测模块。
- 输出回测资金曲线和交易报告。

## Phase 5：交易归因与趋势池稳定性

- 新增交易归因分析。

- 新增前向验证模块。

## Phase 7：Forward Validation

- 冻结策略。
